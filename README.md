# Fabrika

Streamlit-интерфейс для пайплайна генерации гипотез на основе KPI и загруженных PDF/Excel файлов.

## Установка

1. Создайте и активируйте виртуальное окружение.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Настройка `.env`

1. Создайте `.env` на основе `.env.example`.
2. Заполните переменные:

```env
YANDEX_API_KEY=
YANDEX_FOLDER_ID=
YANDEX_BASE_URL=
YANDEX_MODEL=
DEEPSEEK_MODEL=
QWEN_MODEL=
GPT_MODEL=
```

`.env` добавлен в `.gitignore` и не должен коммититься.

## Запуск

```bash
streamlit run app.py
```

## Что умеет интерфейс

- загружать несколько PDF, XLSX и XLS файлов одновременно;
- поддерживать drag-and-drop через стандартный `st.file_uploader`;
- читать каждый файл отдельно и показывать статус обработки;
- объединять контекст из всех успешно прочитанных файлов;
- запускать существующий pipeline без переписывания алгоритмической части;
- работать в демо-режиме без обращения к моделям, чтобы быстро показать пример гипотез;
- показывать отладочную информацию по процессу генерации;
- скачивать PDF-отчет со всеми гипотезами.

## Предупреждения

- При загрузке больших файлов обработка может занимать больше времени, так как системе нужно извлечь из них контекст и передать его в модели Yandex AI Studio.
- Чем больше объем загруженных данных, тем дольше может идти генерация гипотез и итогового отчета.

## Шаги использования

1. Пользователь загружает файлы через интерфейс Streamlit.
2. Пользователь указывает проблему/KPI и ограничения.
3. Система генерирует гипотезы на основе загруженных данных.
4. После генерации пользователь может скачать PDF-отчет со всеми гипотезами.
