# Fabrika

Streamlit-интерфейс для пайплайна генерации гипотез на основе KPI и загруженных PDF/Excel файлов.

## Установка

1. Создайте и активируйте виртуальное окружение.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Настройка `.env`

1. Создайте `.env` на основе `.env.example`.
2. Заполните переменные:

```env
YANDEX_API_KEY=
YANDEX_FOLDER_ID=
YANDEX_BASE_URL=
YANDEX_MODEL=
DEEPSEEK_MODEL=
QWEN_MODEL=
GPT_MODEL=
```

`.env` добавлен в `.gitignore` и не должен коммититься.

## Запуск

```bash
streamlit run app.py
```

## Что умеет интерфейс

- загружать несколько PDF, XLSX и XLS файлов одновременно;
- поддерживать drag-and-drop через стандартный `st.file_uploader`;
- читать каждый файл отдельно и показывать статус обработки;
- объединять контекст из всех успешно прочитанных файлов;
- запускать существующий pipeline без переписывания алгоритмической части.
- время работы программы может быть больше, если пользователь добавит большие по размеру файлы т.к. нейросетям из yandex ai studio тяжело они даются.
- есть демо режим, который не делает обращения к моделям, быстрый способ продемонстрировать, какие гипотезы будет выводить программа.
- есть функции дебага всего процесса.
- возможность скачать pdf отчет со всеми гипотезам.
## Шаги использования
- пользователь отправляет файлы через интерфейс streamlit, записывает проблему/kpi и ограничения.
- после проходит генерация гипотез по этим данным.
- в итоге можно скачать pdf отчет со всеми гипотезами.
